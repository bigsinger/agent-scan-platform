## 统一错误码

- 400 参数格式错误
- 401 未登录或主平台 Token 失效
- 403 无权限或被主平台禁用
- 404 资源不存在
- 409 状态冲突，例如任务运行中不可删除
- 422 字段校验失败
- 423 SQLite 只读或锁等待超时
- 500 未预期错误
